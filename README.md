## My To-Do List App

## Project Description

My To-Do List is a simple and interactive task management web application built using **HTML**, **CSS**, and **JavaScript**. It allows users to add tasks, mark them as completed, delete tasks, and keep track of how many tasks are left to finish.

The project uses JavaScript arrays and DOM manipulation to update the page dynamically without reloading. It also includes bonus features such as preventing duplicate tasks, clearing all tasks, changing the background color, and displaying a celebration message when all tasks are completed.

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as done or undone
- ❌ Delete individual tasks
- 🗑 Clear all tasks
- 📊 Live remaining task counter
- 🚫 Prevent duplicate tasks
- 🎉 Celebration message when all tasks are completed
- 🎨 Change the background color using the color picker

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 💡 Reflection

### What was easy?

Designing the user interface with the provided HTML and CSS was straightforward. Creating the task list and updating the page dynamically became easier once I understood how to use DOM manipulation and event listeners.

### What was difficult?

The most challenging part was managing the task array and making sure the page stayed synchronized whenever a task was added, deleted, or marked as completed. I also had to carefully update the task counter and implement features like duplicate checking and the color picker.

### How I approached the task

I first created the functionality for adding tasks, then implemented rendering using a JavaScript array of objects. After that, I added the Done and Delete buttons, updated the task counter, and finally completed the bonus features such as duplicate prevention, Clear All, the celebration message, and the background color picker.
